# Excel Calendar Date Picker

**Version:** 1.0.0  
**Release Date:** February 16, 2026  
**Compatibility:** Microsoft Excel 2013, 2016, 2019, 2021, Office 365

## Overview

Excel Calendar Date Picker is a professional add-in that provides a visual calendar interface for easy date selection in Excel cells. No more manual date typing - just click and select!

## Features

✅ **Visual Calendar Interface** - Beautiful 175x215 pixel calendar dialog  
✅ **Large Ribbon Button** - Professional calendar icon in the INSERT tab  
✅ **Keyboard Shortcut** - Quick access with Ctrl+Shift+D  
✅ **Smart Date Detection** - Automatically loads existing cell dates  
✅ **International Support** - Works with any regional date format  
✅ **One-Click Selection** - Select today's date with a single click  
✅ **Month Navigation** - Easy browsing through months and years  
✅ **Auto-Format** - Automatically formats selected cells as dates

## Installation

### Automatic Installation (Recommended)
1. Download `ExcelDatePicker.xlam`
2. Double-click the file
3. Excel will automatically install and enable the add-in
4. Restart Excel if prompted

### Manual Installation
1. Download `ExcelDatePicker.xlam`
2. Open Excel
3. Go to File > Options > Add-ins
4. Click "Go" next to "Manage: Excel Add-ins"
5. Click "Browse" and select the downloaded file
6. Check the box next to "Excel Date Picker"
7. Click OK

## Usage

### Method 1: Ribbon Button
1. Select a cell where you want to insert a date
2. Go to the INSERT tab
3. Click the "Date Picker" button (large calendar icon)
4. Select a date from the calendar
5. Click OK

### Method 2: Keyboard Shortcut
1. Select a cell
2. Press **Ctrl+Shift+D**
3. Select a date
4. Click OK

## Troubleshooting

### Add-in doesn't appear in Excel
- Check File > Options > Add-ins > Excel Add-ins
- Make sure "Excel Date Picker" is checked
- Restart Excel

### Ctrl+Shift+D doesn't work
- Verify the add-in is loaded (see above)
- Check for keyboard shortcut conflicts with other add-ins
- Use the Ribbon button instead

### Calendar doesn't show existing date
- Ensure the cell contains a valid date value
- Excel must recognize it as a date (not text)
- Try formatting the cell as Date first
