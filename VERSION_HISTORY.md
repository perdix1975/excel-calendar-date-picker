# Version History

## Version 1.0.0 - Initial Release
**Release Date:** February 16, 2026

### Features
- ✅ Visual calendar dialog (175x215 pixels)
- ✅ Professional ribbon integration with large CalendarInsert icon
- ✅ Keyboard shortcut: Ctrl+Shift+D
- ✅ Smart detection of existing cell dates
- ✅ International date format support
- ✅ Auto-format cells as dates (m/d/yyyy)
- ✅ Month and year navigation
- ✅ "Today" button for quick access
- ✅ Localized interface based on system settings

### Technical Details
- **Platform:** Excel VBA Add-in
- **File Size:** <500 KB
- **Compatibility:** Excel 2013+, Office 365
- **Requirements:** Windows 10 or later

### Known Limitations
- Works with single cell selection only (by design)
- Requires macro-enabled environment
- English interface with international date support
