1. Download the .xlam file
2. Open Excel
3. Go to File > Options > Add-ins
4. Click "Go" next to "Manage: Excel Add-ins"
5. Click "Browse" and select the downloaded file
6. Check the "Excel Date Picker" box
7. Click OK
