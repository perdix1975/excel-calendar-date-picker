# Quick Installation Guide

## For End Users

1. **Download** the `ExcelDatePicker.xlam` file
2. **Double-click** the file to open it in Excel
3. Excel will ask if you want to enable macros - **Click "Enable Content"**
4. **Restart Excel**
5. Look for the **Date Picker** button in the **INSERT** tab

## Using the Add-in

### Quick Start
- Select any cell
- Press **Ctrl+Shift+D**
- Pick a date
- Done!

### Or Use the Button
- Go to INSERT tab
- Click **Date Picker** (calendar icon)
- Select date
- Click OK

## That's It!

You now have a professional date picker in Excel!
