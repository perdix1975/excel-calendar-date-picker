## Submission Process

1. **Create Partner Center Account**
   - Go to partner.microsoft.com
   - Sign up as app developer
   - Pay one-time registration fee ($19 USD for individuals)

2. **Prepare App Package**
   - Final .xlam file
   - README and documentation
   - Screenshots
   - App icon

3. **Submit to AppSource**
   - Go to Partner Center Dashboard
   - Create new Office Add-in submission
   - Fill in listing information
   - Upload app package
   - Upload screenshots and icon
   - Submit for review

4. **Certification Process**
   - Microsoft reviews app (5-7 business days)
   - Security scan
   - Functionality testing
   - Content review

5. **Publication**
   - Once approved, app goes live
   - Available in Microsoft AppSource
   - Searchable in Excel's Get Add-ins menu
